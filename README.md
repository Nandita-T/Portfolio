## Quick Start

1. Clone the repository:

   ```bash
   git clone https://github.com/Nandita-T/Portfolio.git
   ```

2. Open `index.html` in your browser (no dependencies needed!)

Or use a local server:

```bash
python -m http.server 8000
```
